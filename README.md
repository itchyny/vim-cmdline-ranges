# vim-cmdline-ranges
This plugin improves cmdline-ranges for Vim.

Example:
```vim
cmap j <Plug>(cmdline-ranges-j)
cmap k <Plug>(cmdline-ranges-k)
cmap } <Plug>(cmdline-ranges-})
cmap { <Plug>(cmdline-ranges-{)
cmap g <Plug>(cmdline-ranges-g)
cmap G <Plug>(cmdline-ranges-G)
```

Try `:j`, `:}`, or `:gg`.
